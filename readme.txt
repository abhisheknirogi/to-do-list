# To-Do List Web App

A simple and elegant **To-Do List** application built using **HTML, CSS, and JavaScript**.  
It allows users to **add, delete, filter, and mark tasks as completed**, while automatically saving tasks in **localStorage** so they persist even after page refresh.

---

## Features

- Add new tasks with timestamps  
- Delete individual tasks  
- Mark tasks as completed (click on task text)
